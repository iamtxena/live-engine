# Live Engine ⚡

Universal real-time market data engine + paper/live trading bridge
Built to plug directly into Lona.agency (natural-language strategy → live execution)

Supports any asset: BTC, ETH, stocks, ETFs, forex, futures…

### Features
- Multiple assets live feed (Binance, Bybit, Kraken, Polygon…)
- Tick + 1-min historical storage (Supabase)
- Paper trading (testnet)
- Live execution via ccxt (Bybit, Binance, IBKR, Alpaca…)
- Python → TypeScript conversion via Grok AI
- Dashboard with TradingView charts per asset
- Deploy in 2 clicks on Vercel
